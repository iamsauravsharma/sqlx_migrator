pub fn with_prefix(self, prefix: impl Into<String>) -> Result<Self, Error>

Use prefix for migrator table name only ascii alpha numeric and underscore characters are supported for table name. prefix will set table name as _{prefix}{default_table_name} where default table name is _sqlx_migrator_migrations
